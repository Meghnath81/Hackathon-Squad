# Hackathon-Squad
Hackathon Squad Portal – Collaborative Team Matchmaker Dashboard 2025-Present
Personal Project Github
– Developed an interactive dashboard application for matching multi-disciplinary teams during high-throughput
hackathons based on technical skill sets, specialized programmatic roles, and project domain configurations.
– Leveraged persistent client-side status maps to preserve team compositions, participant bio records, and open
project vacancies smoothly across layout updates and browser events.
– Designed a responsive form filtration and search layout using structured arrays, allowing hackers to filter candidate
profiles instantly by skill tags or domain focus.
